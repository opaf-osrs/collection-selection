# Collection Selection

A RuneLite plugin that turns the Collection Log into a challenge mode.

## How it works

- Pick one target item per Collection Log page
- If you get the wrong item first, the page locks permanently
- Pets award unlock points you can spend to buy back locked pages

## Features

- Overlay on the Collection Log highlights your target item and greys out everything else
- Sidebar panel shows your current targets, points balance, and locked pages
- Native OSRS-style popup notification when a page locks
- Per-profile persistence — data follows your account, not your machine

## Installation

Install via the RuneLite Plugin Hub — search for **Collection Selection**.
